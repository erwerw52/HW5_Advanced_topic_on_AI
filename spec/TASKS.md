## Phase 2：核心模組開發 ⚙️

### Task 2.1 - 文檔解析模組 (`document_parser.py`)
- **狀態**：✅ 已完成
- **預估時間**：30 分鐘
- **說明**：實作文檔讀取與結構解析
- **功能**：
  - [ ] 支援 `.txt` 檔案讀取
  - [ ] 支援 `.md` 檔案讀取與解析
  - [ ] 支援 `.docx` 檔案讀取
  - [ ] 自動識別標題層級（H1, H2, H3）
  - [ ] 提取段落與列表內容
- **輸出格式**：
  ```python
  {
      "title": "文檔標題",
      "sections": [
          {"level": 1, "heading": "章節標題", "content": "內容..."},
          ...
      ]
  }
  ```

### Task 2.2 - AI 分析模組 (`ai_analyzer.py`)
- **狀態**：✅ 已完成
- **預估時間**：45 分鐘
- **說明**：使用 Gemini API 分析文檔內容
- **功能**：
  - [ ] 分析文檔主題類型
  - [ ] 推薦適合的模板風格
  - [ ] 提取/生成重點摘要
  - [ ] 將內容整理為 PPT 適合的結構
- **輸出格式**：
  ```python
  {
      "theme": "技術/學術/簡約",
      "slides": [
          {"type": "title", "title": "...", "subtitle": "..."},
          {"type": "content", "heading": "...", "bullets": [...]},
          ...
      ]
  }
  ```

  - [ ] 提供配色、字體、佈局設定
